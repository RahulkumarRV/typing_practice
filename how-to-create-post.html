How to write a blog on This application


You can write any blog by selecting the Create option from the drop-down of the selection action


CREATE:

  • There are four options. The first is (Path Name) which is used for giving slug name (relative URL of the post)
  • second is the Title of the page which is used to give the title of the post which will show on the post
  • the third description of the page which will show on the page, helps SEO optimization of the post
  • after all, you can hit the upload button which pushes your changes to the server
